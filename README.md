# Biomedical-Named-Entity-Recognition

## Feuille de route
1 -> Compréhension du fonctionnement des algorithmes et des hyper-parametres de ces derniers.

2 -> Chargement et expérimentation dans le cadre d'un benchmark des trois, Qutres algorithmes proposés sur les jeux de données listés dans le sujet. 
a. CRF
b. BiLSTM-CRF
c. BERT
d. ~ Spacy (Elmo) ~

le but est de faire une étude comparative pratique sur la détection des différentes entités (Maladies, Gênes, produits Chimiques) 

3 -> Tester les algorithmes entrainés précédemment sur d'autres corpus de thématiques précises et choisis.

4 -> Construction d'un nouveau corpus pour réentrainer les models précédents (en fonction du temps disponible vers la fin)
